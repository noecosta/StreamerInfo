StreamerInfo 0.1.1
This is a simple overview showcasing the different features of the StreamerInfo widget tool. Give it a go and enter a valid Twitch.tv handle into the textbox below.

Generator

Enter a valid Twitch handle between 4 and 25 characters (inclusive).


General API request

URL: -

This is the image generated by StreamerInfo when there are no parameters passed. It automatically sets the mode to automatic and keeps the image quality on the highest setting.
Modern browsers get a WEBP image while older ones are served a JPEG image.

Customized API request (image format/mode)

URL: -

This is the image generated by StreamerInfo when one passes the mode parameter. It generates the image in the desired image format.
Valid values: jpg, jpeg, webp, png, auto.
Default: auto.

Customized API request (quality/compression)

URL: -

This is the image generated by StreamerInfo when one passes the level parameter. It generates the image with the specified quality/compression level.
Valid values: 0 to 9 (inclusive, higher = better, stronger).
Default: 9.
